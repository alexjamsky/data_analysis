# Анализ базы данных сервиса для чтения книг

## Описание проекта
С целью формирования ценностного предложения для нового продукта необходимо проанализировать базу данных крупного сервиса для чтения книг по подписке, содержащую информацию о книгах, издательствах, авторах, а также пользовательские обзоры книг, и сформировать выводы.


## Используемые библиотеки
- *pandas*
- *sqlalchemy*

## Выводы
В ходе исследования было определено количество выпущенных книг с начала 2000 года, а также диапазон количества обзоров и средний рейтинг книг. Также было выявлено издательство, выпустившее наибольшее число книг толще 50 страниц, и авторы с самыми рейтинговыми книгами.
